# 05 — Контент MVP: 3 услуги (Фаза E)

## Цель
Наполнить 3 услуги в Filament-админке. Стандартный формат: 4–6 шагов + FAQ + защищённые файлы.

---

## E1. Стандарт услуги

### Обязательные поля (Service)
- `title` — короткий, описывает суть
- `slug` — человекочитаемый, без кириллицы
- `description_public` — 2–4 абзаца: для кого, что получит, что НЕ входит
- `price` — в центах (€29 = 2900)
- `access_duration_days` — 30
- `is_active` — включить после проверки

### Обязательные блоки контента (MaterialBlock)

| Блок | Тип | Обязательно |
|------|-----|-------------|
| Обзор шагов | `steps` × N | ✅ 4–6 шагов |
| Список документов | `text` | ✅ |
| Практические советы | `tip` | ✅ хотя бы 1 |
| FAQ | `faq` | ✅ 4–6 вопросов |
| Файлы/бланки | `files` | если есть PDF |
| Образцы | `samples` | если есть |
| Блок помощи | `help` | опционально |

### Контент-стандарт

- **Только практическое:** что делать, куда идти, что взять с собой
- **Без истории и законов:** не «В соответствии с законом №...», а «Вам нужно сделать X»
- **Если есть риск ошибки:** короткий блок `tip` (warning) + ссылка на официальную страницу
- **Ссылки на официальные источники** в конце шага или в FAQ
- **Тон:** нейтральный, без обещаний результата

### Структура одного шага (type: `steps`)
```
Шаг N: [Заголовок действия]
[2–4 предложения: что именно делать]
[Если нужно: список документов / адрес / ссылка]
[Если есть нюанс: tip-блок]
```
