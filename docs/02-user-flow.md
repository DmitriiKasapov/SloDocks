# User Flow — SloDocs (MVP)

## Цель
Получить платный доступ к информации по услуге «Оформление ребёнка в школу».

## Основной сценарий

1. Пользователь заходит на главную страницу сайта.
2. Пользователь видит список доступных услуг.
3. Пользователь выбирает услугу «Оформление ребёнка в школу».
4. Пользователь переходит на страницу услуги.
5. Пользователь изучает описание и состав материалов.
6. Пользователь нажимает кнопку «Получить доступ».
7. Пользователь переходит на страницу оплаты.
8. Пользователь оплачивает доступ.
9. Система подтверждает оплату.
10. Система предоставляет доступ к материалам через сайт.
11. Пользователь использует материалы.
12. По истечении срока доступа система закрывает доступ.

## Альтернативный сценарий (ошибка оплаты)

1. Пользователь проходит шаги 1–7 основного сценария.
2. Пользователь пытается оплатить доступ.
3. Оплата не проходит.
4. Система уведомляет пользователя об ошибке оплаты.
5. Пользователь может повторить попытку оплаты или вернуться на страницу услуги.
